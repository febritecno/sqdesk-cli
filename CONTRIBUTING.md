# Contributing to SQDesk

Thank you for your interest in contributing to the SQDesk project!

## How to Contribute

1. **Fork** this repository
2. Create a **new branch** for your feature/fix
   ```bash
   git checkout -b new-feature
   ```
3. Make your changes and **commit**
   ```bash
   git commit -m "Add new feature"
   ```
4. **Push** to your branch
   ```bash
   git push origin new-feature
   ```
5. Create a **Pull Request** to the `main` branch

## Review Process

- All Pull Requests will be reviewed by the maintainers
- Code will be checked for quality and compatibility
- If there are no issues, maintainers will merge it into the `main` branch
- If revisions are needed, you will be notified in the PR

## Guidelines

- Ensure the code is tested before submitting a PR
- Use English for commit messages
- Explain the changes made in the PR description
- Follow the applicable Go style guide (use `gofmt`)

## Questions?

If you have any questions, please open a new [Issue](https://github.com/febritecno/sqdesk/issues).
